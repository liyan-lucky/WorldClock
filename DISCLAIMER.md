# Disclaimer

WorldClock is provided as a general-purpose desktop world clock application.

Time zone data, daylight-saving rules, operating system APIs, runtime libraries, and local system settings may change or produce unexpected results. The application should not be used as the sole source for legal, medical, aviation, transportation, financial trading, emergency, safety-critical, or deadline-critical decisions.

Users should verify important times with authoritative sources when accuracy is critical.

The project does not collect or transmit user data by design. Configuration is saved locally as `world_clock_config.json` unless modified by downstream packagers or contributors.
