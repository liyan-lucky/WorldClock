# Contributing

Thank you for improving WorldClock.

## Contribution rules

Before submitting changes, please confirm:

1. You wrote the contribution yourself, or you have permission to submit it.
2. You did not add secrets, tokens, certificates, account data, private keys, or local user configuration files.
3. You did not add third-party icons, images, fonts, templates, or code without recording the source and license.
4. You checked whether new dependencies have licenses compatible with this project.
5. You tested the relevant implementation before submitting.

## WPF changes

For C# / WPF changes, please build the project with the .NET SDK version declared by the project file.

## Python changes

For Python changes, please document any added dependencies and update `THIRD_PARTY_NOTICES.md` when needed.

## Pull request notes

A pull request should explain the purpose, user-visible changes, test method, and any third-party source used.
