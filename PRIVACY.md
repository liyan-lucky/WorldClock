# Privacy

WorldClock is designed as a local desktop application.

## Data handled by the app

The application may save local settings such as selected cities, theme mode, opacity, window size, window position, seconds display preference, and always-on-top preference.

The default configuration file is named `world_clock_config.json`.

## Network access

The current application design does not require network access for normal clock display. Time is calculated from the local system clock and platform time zone data.

## Data collection

This repository does not include analytics, telemetry, advertising SDKs, account login, remote tracking, or user data upload logic.

## User responsibility

If a downstream distributor modifies the application to add networking, analytics, auto-update, crash reporting, or cloud sync, that distributor should provide its own privacy notice.
